# 人际理解的一种量子力学类比 —— 算符、表象与本征值

**作者：张明楷\ \ 共一：语晴**

---

## 简介

本文借用量子力学的数学结构——算符、本征值、幺正变换与矩阵元——构建一套理解人际差异与包容的类比框架。

- 每个人对应于一个独特的算符，同一事件坍缩出不同本征值
- 理解他人 = 在自身表象下进行幺正变换以求得其本征值
- 矩阵元不可知：本征值告诉你「做了什么」，矩阵元告诉你「如何体验」
- 逆问题不可解：本征值不唯一确定矩阵
- 包容 = 谱分解：识谱、扩谱、容谱
